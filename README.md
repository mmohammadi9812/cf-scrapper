# Code-forces Scraper
---
## Running
use `pipenv` to install dependencies:
```shell
pipenv install
```

to run this project, run:
```shell
pipenv run python scrap.py
```

The site has a throttling rate of 5 requests per second.

more requests might result in a temporary ban.
